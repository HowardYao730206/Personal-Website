## About this project

### Pochita!

Pochita is from the manga/anime series Chainsaw Man. I modeled this piece in Blender and 3D printed it with a Bambu Lab P1S.

You can find the download link below.

This was my first time making such a detailed (to me) model in Blender, and I definitely did not follow every industry-standard modeling practice. Because of that, the model has way more faces/triangles than it needs to have, and the file size is unfortunately very large. Sorry about that! If you know a good way to simplify or optimize it, I would genuinely love to hear from you. Thank you meow :D

*P.S. I am 99% sure the file in the link is not the final version. I currently cannot access the final version that was saved on my other computer, but I will update the file and remove this note when I get the chance. If you cannot wait, please manually increase the size of the tail ring. That should be the only change I made in the final version, and you may need Blender to do it.*
